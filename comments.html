AlbumExposure API

Comments

* All comment operations apply to an album

DEPRICATED: The comments requested as XML currently returns a deprecated representation of the comments which is a string representation of each spread separated by "-----"

GET /albums/#{album.permalink}/comments.xml OR /albums/#{album.permalink}/comments.json
