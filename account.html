Create Account

POST /account.json OR /account.xml

Create an AlbumExposure account. Required fields are:

  • First name
  • Last name
  • Email
  • Login
  • Password
  • Password confirmation
  • Acceptance of AE privacy policy and terms of service
  • JSON request
  • XML request
{
  first_name: "Joe",
  last_name: "User",
  email: "joe@example.com",
  login: "joeuser",
  password: "mypassword",
  password_confirmation: "mypassword",
  company_name: "Joe User Photography",
  terms: "true",
  newsletter: "true"
}
