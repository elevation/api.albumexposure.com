Spreads

* All spread operations apply to an album

Create spread

POST /album/#{album.permalink}/spreads.json OR /album/#{album.permalink}/spreads.xml

Request

Temporarily we recommend that you submit a spread create request as a POST of name-value pairs:

spread[image] = #{Filedata}

Response
Status: 201 Created

Destroy spread

*** IMPORTANT: To replace an existing spread, rather than deleting, just create a spread with the same filename ***

DELETE /album/#{album.permalink}/spreads/#{spread.id}.json OR
/album/#{album.permalink}/spreads/#{spread.id}.xml

Response
Status: 200 OK
